# Task: Implement DataLoader class with CSV/Excel support and encoding detection

## Description

Create the core data loading module with the DataLoader class that handles CSV and Excel files with automatic encoding detection. This includes the main class, encoding detection utility, and module exports.

## Acceptance Criteria

- [ ] Create `src/regression_analyzer/loader/` directory structure
- [ ] Implement `encoding.py` with `detect_encoding()` function using chardet
- [ ] Implement `data_loader.py` with `DataLoader` class
- [ ] CSV loading works with encoding detection and fallback chain
- [ ] Excel loading via pandas/openpyxl with Polars conversion
- [ ] `get_metadata()` returns shape, dtypes, null counts, sample values
- [ ] Create `__init__.py` with proper exports

## Technical Details

### Files to Create
- `src/regression_analyzer/loader/__init__.py`
- `src/regression_analyzer/loader/encoding.py`
- `src/regression_analyzer/loader/data_loader.py`

### Implementation Notes
- Use chardet for encoding detection with 0.7 confidence threshold
- Fallback chain: detected → utf-8 → latin-1 → cp1252
- Excel uses pandas read_excel then converts to Polars
- Handle null values: "", "NA", "N/A", "null", "NULL", "None"
