# Task: Create .dockerignore file

## Description

Create a .dockerignore file to optimize Docker build context by excluding unnecessary files like .git, virtual environments, cache directories, and test files.

## Acceptance Criteria

- [ ] .dockerignore file created in repository root
- [ ] Excludes .git directory
- [ ] Excludes virtual environment directories (.venv, venv, env)
- [ ] Excludes Python cache (__pycache__, *.pyc, .pytest_cache)
- [ ] Excludes test and documentation directories
- [ ] Excludes .claude/ internal configuration

## Technical Details

Files/directories to exclude:
- `.git/` - Version control
- `.venv/`, `venv/`, `env/` - Virtual environments
- `__pycache__/`, `*.pyc`, `*.pyo` - Python bytecode
- `.pytest_cache/`, `.coverage` - Test artifacts
- `tests/` - Test files (not needed in production)
- `docs/` - Documentation
- `.claude/` - Claude configuration
- `*.md` - Markdown files (except important ones)
- `.env*` - Environment files (security)
- `dist/`, `build/`, `*.egg-info` - Build artifacts
