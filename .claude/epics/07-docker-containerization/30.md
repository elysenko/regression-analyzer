## Description

Create a docker-compose.yml file for local development that builds the image, maps ports, mounts volumes for data persistence, and configures environment variables.

## Acceptance Criteria

- [ ] docker-compose.yml file created

- [ ] Volume mounts for /app/data and /app/output

- Service name: `api`
- Build context: current directory
- Ports: 8000:8000
- Volumes: ./data:/app/data, ./output:/app/output

  - OPENAI_API_KEY
  - GOOGLE_API_KEY
  - GROQ_API_KEY
