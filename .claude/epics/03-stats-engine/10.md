# Task: Create stats Pydantic models

## Description

Create the Pydantic result models for the statistics engine module. This includes models for min/max results, regression coefficients and results, feature importance, and the complete statistics report.

## Acceptance Criteria

- [ ] Create `src/regression_analyzer/stats/models.py` with all Pydantic models
- [ ] Implement `MinMaxResult` with min/max values and context
- [ ] Implement `RegressionCoefficient` with p-values and significance
- [ ] Implement `LinearRegressionResult` with metrics and interpretation
- [ ] Implement `FeatureImportance` and `FeatureImportanceResult`
- [ ] Implement `StatisticsReport` for aggregated results
- [ ] All models have proper Field descriptions and validation
- [ ] Models can be imported: `from regression_analyzer.stats.models import MinMaxResult`

## Technical Details

### File to Create
- `src/regression_analyzer/stats/models.py`

### Pydantic Models (per PRD)
- `MinMaxResult`: column, min/max values and indices, context dicts, range/mean/std
- `RegressionCoefficient`: feature, coefficient, std_error, t_statistic, p_value, is_significant
- `LinearRegressionResult`: target, features, coefficients list, intercept, r_squared, adjusted_r_squared, rmse, n_samples, interpretation
- `FeatureImportance`: feature, importance, importance_std, rank
- `FeatureImportanceResult`: target, features list, model_r_squared, method, interpretation
- `StatisticsReport`: minmax list, linear_regression, feature_importance, warnings
