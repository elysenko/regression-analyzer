# Task: Implement min/max analysis and linear regression

## Description

Create the min/max analysis function and linear regression implementation with coefficient statistics including p-values and human-readable interpretations.

## Acceptance Criteria

- [ ] Create `src/regression_analyzer/stats/minmax.py` with `analyze_minmax()` function
- [ ] Min/max includes context from other columns at min/max rows
- [ ] Calculate basic statistics (mean, std, range) for each column
- [ ] Create `src/regression_analyzer/stats/regression.py` with `run_linear_regression()` function
- [ ] Calculate coefficient standard errors, t-statistics, and p-values
- [ ] Generate human-readable interpretation of results
- [ ] Handle edge cases (insufficient data, multicollinearity)

## Technical Details

### Files to Create
- `src/regression_analyzer/stats/minmax.py`
- `src/regression_analyzer/stats/regression.py`

### minmax.py Implementation
- `analyze_minmax(df, columns)`: Returns List[MinMaxResult]
- Get numeric columns if not specified
- For each column: find min/max values and indices
- Capture context (other column values at min/max rows)
- Calculate range, mean, std

### regression.py Implementation
- `run_linear_regression(df, target, features, test_size)`: Returns LinearRegressionResult
- Train/test split with sklearn
- Fit LinearRegression model
- Calculate R², adjusted R², RMSE
- `_calculate_coefficient_stats()`: P-values via scipy.stats.t
- `_interpret_regression()`: Human-readable summary
